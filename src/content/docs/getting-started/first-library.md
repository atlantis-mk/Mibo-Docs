---
title: First Library
description: Add and scan your first media library.
---

Use this guide as the starting point for turning local media into a browsable
Mibo library.

## Prepare media

Choose a directory that contains the media files you want Mibo to scan. Keep
related items together so the scanner can infer meaningful library structure.

## Add a library root

Open the setup or library management view, then add the directory as a library
root. After saving the root, trigger a scan and wait for the scan to complete.

## Verify results

Confirm that discovered titles appear in the library browser and that folder
visibility matches the roles and tags configured for the library.
