---
title: Library Management
description: Manage media roots, scans, visibility, and library organization.
---

Library management is responsible for keeping media roots, scan results, and
user-facing library visibility aligned.

## Media roots

Add a media root for each directory Mibo should scan. A root should represent a
stable ownership boundary for a set of files.

## Scanning

Run scans after adding or changing media. Scan-derived fields should be treated
as owned pipeline data once they enter a pipeline artifact.

## Visibility

Use tags and roles to control which users can see each library area. Avoid
duplicating visibility rules outside the library configuration path.
